ModuleName

Purpose

This operator calculates the main axes of extent in units of mm/cm/m over all given objects connected to inObjectNode. 
Used to get an automatic measure of extent of objects.

Usage

Connect inventor objects to inObjectNode.
Main axes and extents are caclulated automatically and displayed in unites of mm/cm/m.

Details

This module is used in the InterventionPlanner for automatic measureing of any objetcs rendered by module AppVis.
Module AppVis has a inventor output node of a point representation for each rendered object (beside a SSD representation) which is
used as input of module SoMainAxis.

Inputs

Any inventor object.

Outputs

Measurement tool, i.e. the main axes with extent.

Parameters

§         Display
Here you may decide what to display.

o       Input
Display input objects?

o       Transparency:
Transparency of input objects.

o       Axis
Display axes?

o       Origin
Define origin of main axes’s center

o       Unit
Display unit?

o       Name
Display name of the measure tool?

o       Largest Only
Display axis of largest extent only?

o       Bounding Box
Display bounding box

o       Line Width
Line wodth of bounding box

o       Color
Color of tool, axes, units …

§         Measures

o       Axes Orientation
Orientation of main axes

o       Extents
Main extents of connected object(s) in units of mm

o       Axes Center
Center of axes

Example

         Measures the extent of a cubus in combination with a cigar

Tips & Tricks

None

Known Bugs

-         transparency for input doesn’t work
