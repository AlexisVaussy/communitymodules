ModuleName

        SoAngleToObjects

Purpose

Given two input inventor objects the angle is measured between them automatically and displayed in 3D. The angle lines are defined through the largest main axes
of each of the objects.

Usage

See example network.

Details

Inputs

Two arbitrary inventor objects.

Outputs

Output are the 3D angle lines with unit, measure , ….

Parameters

Easy to use.
