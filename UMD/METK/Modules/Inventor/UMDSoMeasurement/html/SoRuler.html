SoRuler

This operator implements a ruler, which can be moved in 3d. The movement can be fulfilled by selecting a part of the tool and move it by dragging, picking or pressing the cursor or page up/down keys.

There is a macro named "Ruler" which defines an additional 2D-Output.

startPos Position of the one end of the angle line in world coordinates
endPos Position of the other end of the angle line in world coordinates
color Color of the tool
lineLength Actual length of the line
minDistance Size of the interruption of the line for the unit placement
keyboardIncrement Step size to move an endpoint by the keyboard
toolNameFlag Display toolname
toolName The toolname
unitFlag Display unit
unit The unit (mm, cm, m)
interactionMode Moving the tool with his dragger or by picking
