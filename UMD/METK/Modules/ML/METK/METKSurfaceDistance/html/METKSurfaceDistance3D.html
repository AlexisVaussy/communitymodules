METKSurfaceDistance3D

Purpose

If you want to know the distance between structures in 3D, you can use the module. For accuracy purposes, the distance should be compute on the original volume data. Therefore, for given objects, the distance should be computed, for example with METKDistanceTransform. The distance to other structures will be computed, if the according entries in the ObjMgr database are set.

Usage

The distance is computed, if the structure is visible (LAY_APPEARANCE, INF_VISIBLE) and if the entry for distance computing is set (LAY_APPEARANCE, INF_SURFACEDIST). When the distance transformation image changes, the distances will be re-computed.

Inputs

The module has one input for the METKManager and one for the distance transformation image.

Outputs

The module has one output for other METK modules and one for the SoIndexedTriangleStripsets containing the distance information.

Parameters

Near/Far Distance

The distance in mm

Near/Far Distance Color

The color used for these distances. Distances between 0 mm and NearDistance mm are colored with NearDistanceColor, and distances nearer then FarDistance are colored with the other one.

Tips & Tricks

The color and/or distance can only be changed before it was computed for the first time for a particular structure.
