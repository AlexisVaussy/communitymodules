METKMeasures

Purpose

This module computes the main axis and volume of a structure as exact as possible. The original segmentation mask is used as input for the main axis computation.

Usage

You can specify a description tag the ObjMgr is searched for.

Details

For computing the main axis as fast as possible, the segmentation mask is opened and converted into a point set. The main axis is computed using this point set.
