FileIterator

Purpose

This module is simply a combination of the following modules: FileDirectory, FileInformation, FieldIterator, MakeName and LoadAny.

Usage

Use the Selection tab to select the files or connect an external list to the Files field in the List tab and use the Iterate tab to loop over the list. Additionally a new name can be created on the Name tab and the file can automatically be read if the EnableOutput field is selected on the Output tab

Details

Beware that the shown line numbers in the List tab start with 1, whereas the iterator starts with 0. This is due to a limitation of the ListView control.

Example

The example network show a loop over the MeVisLab demo data dir.
