FormatFloat

Purpose

You can use this module to convert a number to a string with specified formatting.

Usage

Set the Fromat string, supply the value and check the output.

Details

If a multi number value (like a vector) is supplied the formatting is applied to all components. For the details of the formatting take a look at the Python documentation.

Parameters
