View2DMPRPath

Purpose

This module can be used to view MPR images. It mostly has the same functionality as the View2D module.

Usage

Connect the image to the first input and the path to the second input and select what you want to view.

Details

This module has the same functionality as the View2D module, but is able to show MPR frames or CMPR images defined along a given path. The shown image have the correct world matrix which makes them suitable for overlays and position selection like drawing CSOs or XMarkers. The connected path can be either a KeyFrameList or a XMarkerList. In the latter case the markers are converted to a KeyFrameList using the PathToKeyFrame module. The settings of this module can be specified in the Settings window. This window can also be accessed by pressing S when the cursor is in the viewing area. Note that only the fields listed below are persistent. Using the mouse wheel you scroll through the frames or change the path angle of the CMPR depending on the current shown content

Input

  • input0 Image to be reformatted
  • path Path defined by a XMarker or KeyFrame List
  • inInvPreLUT Inventor input before LUT and viewer inventor input, e.g. for overlays independent of LUT settings
  • inInvPostLUT Inventor input between LUT and viewer inventor input
  • inInvPostViewer Inventor input behind viewer inventor input, e.g. for diagram overlays

Output

  • mprSliceImage Selected MPR frame
  • stack Stack of all MPR frames. Note that this stack contains an average world matrix, which mostly will be incorrect
  • cmpr CMPR image along the selected angle
  • self Shown image
  • scene Inventor scene showing the selected frame

Parameters

  • inventorInputOn Show the inventor input connectors
  • showOutputs Show the output connectors
  • timePoint Current shown time point
  • currentFrame Selected MPR frame
  • currentPlane Plane of the current MPR frame
  • rotation Rotation of the current MPR frame
  • translation Translation of the current MPR frame
  • worldPosition World position of the cursor
  • pathAngle Angle of the CMPR image
  • fieldOfView Shown field of view
  • outputSize Voxel size of the shown image
  • cursorAlpha Alpha of the cursor
  • viewContent Select MPRFrame, CMPR or Inventor scene to be shown in the viewer
