nSyncViewer

Purpose

This module can be used to synchronize up to 20 viewers and has an inventor input for each of them.

Usage

Set the numberOfInputs and connect the inputs. Use the mouse cursor to sync the different viewers.

Details

This module creates a network containing a OrthoView2D module for each viewer and synchronizes the world positions of all viewers. Additionally the sliceZoom, the LUTSettings and the timepoints of all viewers can be synchroized. It is also possible to reformat all images to a selected reference input image. Pressing 'S' will show a window with all parameters.

Parameters

  • numberOfInputs The number of shown input connections
  • showInventorInputs Show a inventor connection for eacht input
  • singleInvInput The used inventor input of the internally used OrthoView2D modules, i.e. inInvPreLUT, inInvPostLUT or inInvPostViewer
  • layout Layout string. This should have the format of a Python dictonary, i.e. {#Inputs:[#rows,#columns],#Inputs:[#rows,#columns]} where # indicates "numberOf". For example {5,[3,2]} will generate 3 rows and two columns when numberOfInputs = 5. The first and second row will have 2 viewers, the third will have one viewer and one grey area. The size of all viewers will be the same. If the layout for the current number of inputs is not specified, the module will try to make as many rows as columns
  • syncZoom Synchronize the slice zoom
  • syncLUT Synchronize the GreyCenter and GreyWidth
  • syncTimepoints Synchronize the time points
  • reformat Reformat all input images to a specified reference image
  • referenceGrid The reference input for the reformat
  • worldPos Current cursor position in world coordinates
  • coordSystem Coordinate system of the shown coordinate, i.e. Voxel or World
  • interpolationMethod Interpolation method for the reformat operatation
  • cursorAlpha Alpha value of the shown cursor

Example

The example network show 2 images with two overlays simultaneously.

Tips and Trics

Pressing 'S' on any viewer will show the global settings of the module. Pressing 'F' on a viewer will show the settings of that viewer.
