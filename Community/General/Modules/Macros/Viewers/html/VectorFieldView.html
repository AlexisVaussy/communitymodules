VectorFieldView

Purpose

This module show a vector field as a mesh or vector lines overlayed on a scalar image

Usage

Connect a scalar image to input0 and a vector image to input1 and select what you want to see.

Details

If you press the Settings button you can specify all options of the used SoView2DDeformationGridView, SoView2DVectorFieldView and SoView2DOverlay modules. Additionally you can toggle the scalar input on and of and select to show the vector values at the cursor in stead of the scalar values. Using Shift-Right mouse button, you can change the LUT values of the magnitude overlay.

Example

The example network show a vector overlay of a volume ramp image. Just scroll through the slices and change some settings to see the effect.
