SetReSet

Purpose

This module can be use to see if some Field in a network has changed.

Usage

Press Set to Set the value. Press Reset to reset it

Details

The OnTrigger, the OffTrigger and the Int value are only triggered when the value changes. Value is set to True every time Set is pressed.

Parameters

  • value Bool value set on pressing Set
  • intValue Int value set to 1 when Set is pressed the first time and to 0 when Reset pressed for the first time. This field only changes if Value changes.
  • set Set value
  • reset ResetValue
  • OnTrigger Triggered when switching to the value to On
  • OffTrigger Triggered when switching to the value to Off
