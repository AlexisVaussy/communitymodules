FieldTrigger

Purpose

You can use this module to trigger fields in a specified order.

Usage

You can select the fields in the left tree view of all modules in the current network. Double clicking or pressing the right arrow will select the field and add it to the selected fields view in the right panel. You can also simply drag and drop a field on the GUI. Double clicking a field in the right panel or pressing the left arrow removes the field from the list. Using Shift-Up/Down you can move a selected field up and down in the list.
You can specify a sleep time in seconds, which is the time between each trigger (if AfterEachTrigger is selected) or between two repetitions.
After each trigger the following commands are issued:
MLAB.processEvents()
MLAB.processInventorQueue()
Triggering is done with a touch() on the selected fields.

Details

Pressing the round arrow will update the field tree in the left panel.

Parameters

  • allFields Check to see a list off all fields in the network. If not checked only the trigger fields are listed.
  • selectedFields List of selected fields.
  • start Start triggering the selected fields.
  • stop Stop the triggering.
  • sleep Waiting time in seconds.
  • sleepEachStep Wait after each trigger. If not checked wait after each sequence.
  • repetition Repeate the trigger sequence.
  • currentIteration Current iteration.
