Purpose    Usage    Details   

Purpose

Compare modules can be used to compare the field values of two modules in the network or to compare the field values of a module with the default values.

Usage

Select the two modules to see their parameters and their differences. To compare a module with it's default field values, select the "Default" module for the other module.

Details

Pressing "Compare" will update the comparison with the current field values.
Pressing "Update module list" will update the list of selectable modules with those in the current network.
The module only shows fields which the two selected modules have in common. It does not show any trigger fields nor instanceName fields.
