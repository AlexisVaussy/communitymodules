Transformix

Purpose

This module creates a command line call to the Transformix image transformation executable. The input image is saved to the temporary directory. Transformix is called to work on those files, and writes its results in a subdirectory './Transformix'. Consequently, this module reads the Transformix transformation results. Please refer to the Elastix website (http://elastix.isi.uu.nl) for the documentation of the Transformix program and all its parameters.

Usage

Provide the transformix executable, a working directory and a transformation file. Input a a moving image and press 'Update' to start Transformix. Use any of the other parameters to deviate from default behaviour.

Make sure that if you use multiple instances of this module, that they all have their own working directory!

Details

For detailed documentation of the Transformix parameters, visit http://elastix.isi.uu.nl.

Inputs

Elastix has one mandatory input.

input0
The input image that will be transformed.

Outputs

Transformix has one mandatory output, and three optional outputs.

outputImage
The transformed image, defined in the fixed image domain, as defined by the transformation file.

outputDeformationField
A vector image, defined in the fixed image domain, that for each voxel at position p_f contains a relative position vector v in world coordinates pointing to the corresponding position in the moving image (p_m):
p_m = p_f + v

outputJacobian
A float image, defined in the fixed image domain, that contains the determinant of the Jacobian of the transformation. Values smaller than 1 indicate local compression, values larger than 1 indicate local expansion, and 1 means volume preservation. Generation of this image can be enabled or disabled.

outputFullJacobian
The internal name is outputFullJacobian, the type is a macro Image.
A matrix image, defined in the fixed image domain, that contains the spatial Jacobian of the transformation. Generation of this image can be enabled or disabled.

Parameters

The parameters for the transformation itself should be provided through a parameter file, that can be generated using Elastix.

Tranformix Executable
The Transformix executable. It can be downloaded from http://www.isi.uu.nl.
If the directory that contains Transformix has been added to the path, just the transformix executable filename suffices.
Default is transformix.exe.

Working Directory
The directory in which the input and output images, and the transformix log and parameter files will be saved.
Default is C:/temp/.

Transformation File
The parameterization of the transformation. Typical use of this module links the 'Resulting transformation file' of an Elastix module to this field.

Auto Update Transformation File
On change of the working directory, the transformation file will be automatically searched.
Default is FALSE.

Create Image
Toggle creation of the transformed output image on/off.
Default is TRUE.

Create Deformation Field
Toggle creating the defrmation field output on/off.
Default is FALSE.

Create Jacobian Field
Toggle creating the Jacobian image output on/off.
Default is FALSE.

Create Jacobian Matrix Field
Toggle creating the Jacobian matrix image output on/off.
Default is FALSE.

Update
The internal name is update, the type is a macro Trigger.
Call transformix (command line call).

Delete Files
Experimental. Delete intermediate files generated by Transformix and this module.

Events And Interaction

This module may be used together with the ElastixParameterFile and Elastix modules.
