ElastixParameterFile

Purpose

This module reads and writes parameter file for the Elastix registration program. Please refer to the Elastix website (http://elastix.isi.uu.nl) for the documentation of that program and all its parameters.

Usage

Load an existing parameter file to start with parameter settings in that file, or just start configuring your parameters from scratch from the default parameter settings of this module. Once all parameters have been configured, save the file by assigning the output parameter filename and pressing 'Save'.
The main components of the registration framework can be selected in the 'Main' group. To configure each of these components, use the tabs below.

Details

For detailed documentation o the Elastix parameters, visit http://elastix.isi.uu.nl.

Parameters

Output parameter filename:
The parameter file to write the current parameter settings to.

Save
(Over)Write the output parameter file with the current parameter settings.

Input parameter filename:
The parameter file to read the parameter settings from.

Load
Update the parameter configuration with the settings defined in the input parameter file.

Events And Interaction

The parameter file field can be linked to the input parameter field(s) of the Elastix module, to run elastix directly from within MeVisLab. Additionally, use the Transformix module to employ the resulting transformation.
