CallApplication

Purpose

To create a commandline call to an application

Usage

Select an executable, optionally provide command line parameters and a working directory and press 'Call' to execute. optionally print command line output in the log window.

Details

Only tested for Windows. Many thanks to developers from BIGR (http://www.bigr.nl) for the initial version of this module.

Inputs

None.

Outputs

None.

Parameters

The only required parameter is the executable.

Executable
The internal name is executable, the type is a macro String.
The application to be called.

Arguments
Optional command line arguments for the application.

Working Directory
The temporary directory to work in.

Output To Log Window
Output command line program messages to the log window.
Warning: The log window processing is really slow for large messages. Disable this feature is not necessary.
Default is TRUE.

Call
Execute the command line call.

Log Window
Prints program messages if activated by 'output to log log window'.

Clear
Clear the log window.

Call Succeeded
Indicator that shows whether the call was successful. If the executable does not exists, or if anything else fails it will be False.
Default is FALSE.
