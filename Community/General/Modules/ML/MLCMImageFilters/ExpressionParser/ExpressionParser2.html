ExpressionParser2

Purpose

More generic Arithmetic2 (but a bit slower ...).

Usage

Connect 2 images, and specify the expression. In the expression, x refers to voxel value of first image, and y to voxel value of second image. Unary minus is not implemented, replace with e.g. (0-1)* instead of -.

Input

  • input0 First input image (x in expression)
  • input1 Second input image (y in expression)

Output

  • output0 Result of evaluating expression for each voxel.

Parameters

  • expression Expression to parse. x stands for the first input image, y for the second input image. See NewImageFromExpression for details on the expression syntax.
