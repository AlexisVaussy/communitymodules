ExpressionParser1

Purpose

Generate an image where the voxel value is the result of evaluating an expression for that voxel. Use x in the expression for the value of the corresponding input image voxel.

Usage

Specify an expression (where you can use for x the input voxel value).

Details

The expression does not handle unary minus. E.g. the following is not a valid expression:

-x

The following expression can be used as a substitute:

(0-1)*x

Expressions that can not be parsed will not give an output image.

Input

  • input0 Input image.

Output

  • output0 Output image.

Parameters

  • expression Expression to parse. x stands for the input image. See NewImageFromExpression for details on the expression syntax.

Example

Example expressions:
  • Add 10 to the image: x + 10
  • square the image: x*x
  • quadratic exponential decay: exp( (0-1)*(x*x) )
