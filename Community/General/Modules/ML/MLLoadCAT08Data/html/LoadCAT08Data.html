LoadCAT08Data

Purpose

The LoadCAT08Data loads the training data for the Rotterdam Coronary Artery Algorithm Evaluation Framework, a.k.a the Coronary Artery Tracking
competition that was carried out as a workshop at the MICCAI'08 conference. The data can be downloaded here.

Usage

See the example network for the best usage of the module.

Details

See the documentation of the data on the page for the Rotterdam Coronary Artery Algorithm Evaluation Framework.

Inputs

-

Outputs

The module outputs the following:

XMarkerLists containing the
Points A
Points B
Points S
Points E

An XMarkerList with all center line points (the type of the XMarkers is set to the vessel number).
An ML Graph structure which can be visualized with the SoVascularSystem module.

Parameters

The .mhd file for which the data should be loaded (see the example network).
