TestFilter

Purpose

The module allows to select a single curve from a collection of several curves (CurveList) and therefor splits the input CurveList into (1) the selected curve, (2) the curves preceding the selection and (3) the remaining curves.

Usage

Just connect a module providing a CurveList and then select the desired curve using the GUI

Details

-

Inputs

There is only one input for the required CurveList.

Outputs

The module provides three outputs:
(1) outCurveList1 - the list of curves from the input CurveList beginning from index 0 till the selected curve. The output will be empty if the user selects the first provided curve. Selection of the last curve in the list will result in an output with n-1 curves.
(2) outCurveData - the curve that has been selected by the user.
(3) outCurveList2 - the list of curves from the input CurveList beginning after the selected curve till the end. The output will be empty if the user selects the last provided curve. Selection of the first curve in the list will result in an output with n-1 curves.
