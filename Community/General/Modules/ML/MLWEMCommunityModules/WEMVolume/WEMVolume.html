WEMVolume

Purpose

Computes the volumes of the WEM patches and determines the min and max volume and a volume curve.

Usage

Connect WEM.

Input

  • inWEM Input WEM.

Output

  • outputVolumeCurve Output volume curve.

Parameters

  • minVolume Minimum patch volume.
  • maxVolume Maximum patch volume.
  • minVolumePatchIndex Index of patch with minimum volume.
  • maxVolumePatchIndex Index of patch with maximum volume.

Example

See example network.
