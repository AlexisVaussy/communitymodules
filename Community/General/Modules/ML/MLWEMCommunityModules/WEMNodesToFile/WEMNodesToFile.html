WEMNodesToFile

Purpose

Outputs the node coordinates of the input WEM to a textfile

Usage

Connect WEM and press save.

Details

Every line in the file represents a point. Coordinates are separated by spaces. When transformix option is enabled two extra lines are added at the start of the file. One with the text 'point' and one with the number of points.

Input

  • inWEM Input WEM.

Parameters

  • filename Output filename.
  • transformixCompatible Turn this option on if you want to output transformix compatible point files.

Example

Not available.
