CSOPointsToXMarkers

Purpose

Output the points of a CSO in an XMarkerList. CSO normals can be computes as well and are then outputted in the vector element of the markers. The marker type field is filled with the CSOListIndex, unless setIdInMarkerType is checked, in which case the marker type field is filled with the CSOId.

Usage

Connect CSOList, enter index of CSO to output and change settings.

Details

Normals are computed by computing the outer product of the plane normal in which the CSO lies and the vectors between the CSO points.

Input

  • inputCSOList Input CSOList

Output

  • outputXMarkerList Output XMarkerList

Parameters

  • listIndexFld Index of CSO in CSOList to convert (-1 = all)
  • convertPathPoints Enable/disable conversion of pathpoints
  • outputCSONormals Enable/disable computation and output of CSO normals in vector elements of XMarkerList
  • setIdInMarkerType Checking this will make the CSOId appear in the marker type field (instead of the default CSOListIndex).
