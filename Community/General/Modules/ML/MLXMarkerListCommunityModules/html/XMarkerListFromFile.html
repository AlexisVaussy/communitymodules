XMarkerListFromFile

Purpose

Load an XMarkerList from a plain text file.

Usage

Choose filename, select elements of the XMarkers to load, select coordinate system and press load.

Details

Elements are read in the order they are presented in the GUI. So position-x is read before position-y, etc. Random ordening of the elements in the text file is therefore not possible. When the voxel coordinate system is selected an input image need to be connected. Elements are read based on any white space separation. Other separators are not supported.

Input

  • input0 Input image needed for voxel to world matrix when the input coordinate system is set to voxel.

Output

  • outputXMarkerList The loaded XMarkerList

Parameters

  • filename Filename of text file to read.
  • importPositionX Import position x-coordinate.
  • importPositionY Import position y-coordinate.
  • importPositionZ Import position z-coordinate.
  • importPositionT Import position t-coordinate.
  • importVectorX Import vector x-coordinate.
  • importVectorY Import vector y-coordinate.
  • importVectorZ Import vector z-coordinate.
  • importType Import type attribute.
  • inputCoordinateSystem Coordinate system of input file.
  • numElementsToSkip Number of elements to skip at beginning of file.
  • update Start loading XMarkerList.
  • autoUpdate Automatically load XMarkerList on any field changes.
