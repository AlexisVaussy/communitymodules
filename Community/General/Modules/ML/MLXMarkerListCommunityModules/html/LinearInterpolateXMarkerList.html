LinearInterpolateXMarkerList

Purpose

Linearly interpolates an XMarkerList. Markers are interpolated between the input markers using the provided sample distance.

Usage

Connect an XMarkerList and provide a sample distance in the GUI.

Details

Note that the input markers are not perse part of the output XMarkerList as the distance between the input markers is not always a multitude of the sample distance.

Input

  • inputXMarkerList Input markers

Output

  • outputXMarkerList Equidistantly linearly interpolated XMarkers

Parameters

  • sampleDistance XMarker sample distance in mm. Markers are equidistantly linearly interpolated between the input makers using this distance.
