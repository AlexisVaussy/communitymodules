CurveFilter

Purpose

This module filters out curves from a curvlist based on the given min and max index. Additionally it can crop all curves between two indices.

Usage

Connect the input curve list,specify the first and last index and press Update or select "Auto update".

Details

The min and max of the Set specify the first and last CurveData object from a CurveList object. If one of the CurveData objects from the CurveList has more than one series defined, you can choose the first and last index of the Series of all CurveData objects in the list. If all CurveData objects only have one series defined, you cannot choose the Series.
The output curves can also be croped to a specified start and end index.
If "Auto update" is selected, the module reacts on all input and parameter changes

Input

  • inCurveList Input CurveData or CurveList object

Output

  • outCurveList Filtered CurveList

Parameters

  • update Update the output
  • autoUpdate Auto update on input or parameter changes
  • minCurveSet First curve set to be copied to the output
  • maxCurveSet Last curve set to be copied to the output
  • minCurve First curve to be copied to the output
  • maxCurve Last curve to be copied to the output
  • cropCurve Crop all autoput curves
  • start First index of the croped curve
  • end Last index of the croped curve
