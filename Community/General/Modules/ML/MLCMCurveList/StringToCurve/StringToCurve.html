StringToCurve

Purpose

This module can be used to generate a curve or multiple curves by splitting a string based on seperation characters for curves and values

Usage

Simply supply the string and specify the splitting characters

Output

  • outCurveList CurveList containing the generated curves

Parameters

  • curveString String containing all the curve data
  • indexString Optional string with the indices of all the curve points
  • valueSeparator Characters that split the curve into values
  • curveSeparator Characters that split the string into curves
