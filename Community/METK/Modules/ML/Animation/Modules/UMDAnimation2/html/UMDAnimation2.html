TestFilter

Purpose

The TestFilter module offers operations to remove edges from images. It also can fill regions in 2D and 3D, overwrite it by simple text or boxes, lines, pixels, spheres, circles, cones, halfspaces, ellipses or disks. Coordinates for object drawings are received by a field connection and are used to specify the objects' location, orientation and extents. All pixels values written into an image are not simply overwritten but replaced by the result of user selectable function. Undo and redraw facilities exist which also permit changes of object properties after drawing them.

Usage

Press button 1, then button 2 and hope that it does not explode!

Details

The Draw module and the SimpleDraw macro offer operations to draw objects into images. Regions can be filled in 2D/3D, overwritten by simple text.

Inputs

The module has one input which must be of an integer data type. Floating point data cannot be handled and will result in an empty output image.

Outputs

The module has two outputs. The first is the normalized image and the second is the inverted image.

Parameters

Main

This panel contains the frequently used drawing parameters.
  • Draw:
    Selects the object to be drawn, default is Nothing.
    • Nothing: Changes of Position Input do not cause any manipulation of the image.
    • Voxel: At the position given by Position Input a voxel of the size VoxelSize is drawn.
    • Line: At the position given by Position Input a line of width VoxelSize is drawn.
  • Undo:
    Restores the image with all information saved in the undo list.

Text

This panel contains the special parameters used for text drawing.
  • Text:
    Selects the object to be drawn, default is Nothing.
    • Nothing: Changes of Position Input do not cause any modification of the image.
    • Voxel: At the position given by Position Input a voxel of the size VoxelSize is drawn.
    • Line: At the position given by Position Input a line of width VoxelSize is drawn.
  • Pixel Size:
    Selects how the object is drawn, default is 10.
