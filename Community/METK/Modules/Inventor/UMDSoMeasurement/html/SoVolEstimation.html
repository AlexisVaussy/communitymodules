SoVolEstimation

This operator implements a shape, which can be transformed in 3d. The movement can be fulfilled by selecting a part of the tool and move it by dragging, picking or pressing the cursor or page up/down keys.

There is a macro named "VolumeEstimation" which defines further functionalities and an additional 2D-Output.

startPos Position of the shape's center in world coordinates
rotation Rotation of the shape
scaleFactor Scale factor of the shape
color Color of the tool
volExternFlag Use an extern measure
externVolume The extern measure
keyboardIncrement Step size to move an endpoint by the keyboard
toolNameFlag Display toolname
toolName The toolname
unitFlag Display unit
unit The unit (mm³ (cmm), ml, l)
interactionMode Moving the tool with his dragger or by picking
rasterFunction Select a shape (ellipsoid, cuboid, cylinder)
