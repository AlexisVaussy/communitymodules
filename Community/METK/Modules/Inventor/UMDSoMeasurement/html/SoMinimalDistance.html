ModuleName

Purpose

Calculates the minimal distance between two inventor node objects. Very fast. Used in the InterventionPlanner to get distances between tumors and vessels , for example.
See also:  "Integration automatischer Abstandsberechnungen in die Interventionsplanung", Bernhard Preim, Christian Tietjen, Milo Hindennach, Heinz-Otto Peitgen, Bildverarbeitung fuer die Medizin (Erlangen, Mars 2003), Springer, pp. 259-263 .

 

Usage

Connect two different inventor node objects to inputs inObjectNode and inObject2Node. The minimal distance between them is calculated automatically
and optional displayed as a distance line and its extent/unit within the 3D scene.

Details

See publication.

Inputs

Any inventor objects.

Outputs

Measurement tool, measure result.

Parameters

§         Display
Here you may decide what to display.

o       Input
Display input objects?

o       Transparency
Transparency of input objects.

o       Tool
Display distance line? This is not a real parameter, i.e. its true always.
This is for symmetry with the SoMainAxis module.

o       Unit
Display unit?

o       Name
Display name of the measure tool?

o       Display on valid
Distance line is visible on valid measurements only, i.e. two objects are connected to the inputs.

o       Color
Color of the distance line, the extent …..

§         Measures

o       Distance
Minimal distance between the two input objects in mm.

o       Error
Describes the maintainable error between 0% and 100%. The default value is 0%.

o       Start Pos/End Pos
Coordinates of the nearest points. Start Pos: point on inObjectNode, End Pos: point on inObject2Node .

§         Advanced

See publication.

Example

         Measures and displays the minimal distance between two clones which are translated/rotated to each other. Changing extent of translation, the measure
            is adjusted automatically.
